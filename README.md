# EMIL — Тренажёр 🧠⚡

Персональный геймифицированный тренажёр для ежедневных занятий: **веб-разработка** (HTML/CSS/JS) и **английский язык A2→B1**.

**▶ Живая версия:** https://abdullinemil6901-crypto.github.io/learning-hub/

## Что внутри

- 📚 **Уроки программирования** — теория, викторины, практика с кодом и живым превью
- 🇬🇧 **Уроки английского** — грамматика, упражнения, работа над ошибками
- 🃏 **Карточки слов (SRS)** — интервальное повторение, чтобы слова запоминались прочно
- 🔥 **Стрик и XP** — день засчитывается от 30 XP, каждые 7 дней — заморозка стрика
- 🗺️ **Путь до junior** — дорожная карта модулей с теорией, тестами и запуском кода в живом превью
- 🪙 **Магазин и коллекция** — монеты за мастерство, усиления (заморозка стрика, x2 XP) и коллекционные карточки персонажа
- 📊 **Дашборд** — тепловая карта активности за полгода, общий прогресс, ежедневные квесты

- ☁️ **Облачная синхронизация** — вход по ссылке на почту (Supabase), прогресс доступен с любого устройства
- 📱 **PWA** — ставится на главный экран телефона как приложение, работает офлайн

## Как это устроено

Всё приложение — один файл `index.html` без сборки (шрифты и иконки подгружаются с CDN). Прогресс всегда хранится в `localStorage` браузера; после входа по почте (кнопка-облачко в шапке) он дополнительно синхронизируется с Supabase и подтягивается на любом устройстве. Экспорт/импорт в файл остаётся как резервная копия.

## Мобильная версия

Откройте сайт на телефоне → в меню браузера выберите «Добавить на главный экран» — тренажёр установится как приложение: свой значок, полный экран, офлайн-режим.

## Картинки коллекции

Карточки в разделе «Коллекция» подгружаются из `assets/collection/` по именам из массива `NFTS` в `index.html` (например `assets/collection/SHIMPA V SPALNE.jpg`). Пока файла нет — карточка показывает аккуратный плейсхолдер. Загрузите арты в `assets/collection/` с точными именами, чтобы коллекция ожила.

## Запуск локально

Просто откройте `index.html` в браузере — сервер не нужен.

## Публикация (GitHub Pages)

Репозиторий готов к публикации: **Settings → Pages → Source: Deploy from a branch → Branch: `main` / root → Save**. Через минуту сайт появится по ссылке выше.
